## CSS

|key|function                                                |
|---|--------------------------------------------------------|
|vic|Visually select inner CSS (inner rules)                 |
|vac|Viscually select "all" CSS (entire block w/ selector(s))|
|cic|Change inner CSS                                        |
|cac|Change "all" CSS                                        |
|dic|Delete inner CSS                                        |
|dac|Delete "all" CSS                                        |
