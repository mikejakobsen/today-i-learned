To list the comments, which are on hold.
```
wp comment list --status=hold --field=ID
```

To delete them
```
wp comment delete $(wp comment list --status=hold --field=ID)
```

And then trash them
```
wp comment delete $(wp comment list --status=trash --field=ID)
```
